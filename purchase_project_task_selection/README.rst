===============================
Purchase Project Task Selection
===============================

Este modulo permite vincular lineas de compra con proyectos y tareas.

Caracteristicas
===============

* Agrega los campos Proyecto y Tarea en cada linea de compra.
* Filtra las tareas por proyecto.
* Mantiene consistencia entre proyecto y tarea.
* Agrega un boton en la tarea para ver las lineas de compra vinculadas.

Nota sobre analitica
====================

Este modulo no modifica automaticamente la cuenta analitica ni la distribucion
analitica de las lineas de compra.
