# Dependencias de Etapas en Tareas

1. Instala el modulo y abre cualquier tarea del proyecto.
2. En la pestana **Dependencias** selecciona las tareas que se recomienda
   completar antes de mover la tarea actual.
3. Define la etapa obligatoria con "Etapa obligatoria de cierre"; al cambiar de
   etapa se mostrara una advertencia si hay bloqueantes pendientes y solo al
   llegar a esa etapa se permitira cerrar la tarea.
